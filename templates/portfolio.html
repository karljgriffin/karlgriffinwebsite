Automated Gym Booker (Web Browser Automation)

During the COVID-19 restrictions, the gym in the university in which I am studying only allocated a certain number of places for each gym slot, with booking opening exactly three hours in advance of the slot. As a result, it booked out almost instantly and people often didn't get a slot. This simple script automatically booked the specified gym slot - all I had to do was run the script just moments prior to the booking time. The functionality is enabled by employing Selenium for the browser automation, and my StudentID was stored as an environment variable and was accessible using os.environ.

Premier League Stats (Web Scraping, Data Analysis & Visualisation)

This is a project which I undertook because of my love for football. I wanted to work on a Premier League (PL) data science project but all of the API's were too expensive. So this project scrapes a popular football stats website (FotMob) using Beautiful Soup, and I then perform some structuring of this scraped data and convert it into a Python dictionary and subsequently a .JSON format (just paste the whole .JSON file into JSON Viewer to inspect all of the scraped data). The data that I have accrued comprises the following:

• Includes every season from 2016/17 to 2021/2022
• Includes every team that has played in the PL during that period
• Includes 26 different stats

Once I had converted the data into a .JSON format, I was then able to make a Pandas dataframe (df) which I indexed by year and then by club. I performed some further analysis on the data using Pandas, and employed Matplotlib to visualise some of the stats. I also saved the data to a SQLite database.
